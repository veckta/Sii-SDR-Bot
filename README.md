# Sii-SDR-Bot 🤖

Bot de **WhatsApp** integrado con Inteligencia Artificial diseñado para la calificación de leads, diagnóstico técnico y automatización operativa. Desarrollado como backend (SII Chatbot Ultra) para el ecosistema **SII** ([siiarg.com](https://siiarg.com)).

## 🚀 Características Principales
- **Integración con WhatsApp (Meta API):** Recepción y envío de mensajes y audios en tiempo real.
- **Análisis Inteligente y SDR:** Respuestas automatizadas y clasificación de clientes (Abono, Alta Escala, Rechazo) mediante OpenAI / Groq.
- **Procesamiento de Audio:** Transcripción y generación de notas de voz usando Whisper y TTS.
- **Base de Datos:** Registro asíncrono e historial de sesiones integrado con Supabase.

## 🛠️ Stack Tecnológico
- **Core:** Python, FastAPI, Uvicorn
- **Integraciones:** Meta WhatsApp API, OpenAI API, Groq, Supabase
